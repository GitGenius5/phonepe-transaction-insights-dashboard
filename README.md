# PhonePe Transaction Insights Dashboard (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard that analyzes digital payment trends across states and districts in India using the PhonePe Pulse dataset.

The dashboard helps understand transaction growth, user activity, and regional performance through clear and structured visualizations.
